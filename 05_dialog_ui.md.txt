# TC-05: Проверка UI окна «Район (создание)»

**Приоритет:** High  
**Тип тестирования:** UI / Функциональное  
**Окружение:** Windows 10, Yandex 25.10.3.1021, 1920x1080  
**Автор:** Антропова Ольга

## Предусловия

1. Открыт сайт: https://demo.app.stack-it.ru/fl/.
2. Выполнена авторизация под учётной записью:
   - Логин: DEMOWEB
   - Пароль: awdrgy
3. Открыт раздел: «Адресный фонд», «Адреса проживающих»
4. Таблица загружена, тулбар с иконкой «Плюс» отображается

## Шаги

1. Нажать на иконку «Плюс» в тулбаре таблицы.
2. В выпадающем списке выбрать значение «Район».
3. Проверить отображение диалогового окна «Район (создание)».
4. Проверить наличие и состояние элементов окна:
   - Заголовок окна и таблицы
   - Полей ввода «Название района» и «Номер в списке»
   - Кнопка «Внести»
   - Кнопка «Отмена»
   - Иконка «Закрыть» (крестик)
   - Кнопка «Добавить запись»
   - Кнопка «Поиск»
   - Кнопка «История»
   - Чек-бокс
   - Колоноки Парметр, Значение, Дата начала, Дата окончания таблицы Параметр
   - Кнопка настроек таблицы
   - Строка пагинации 
5. Проверить поведение кнопок без ввода данных.
6. Проверить всплывающие подсказки с названием кнопок.
7. Проверить закрытие окна:
   - через кнопку «Отмена»
   - через крестик  
8. Проверить фокус и подсветку полей ввода.
9. Проверить поведение кусрора при наведении элементы окна 
10. Работа кнопки «Добавить запись»
11. Работа кнопки «Поиск»
12. Работа кнопки «История»
13. Работа чек бокса
14. Работа кнопки настроек таблицы
15. Работа фильтрации в колонках таблицы
16. Работа строки пагинации
17. Наличие статичного текста в окне


## Ожидаемый результат

Шаг 3:
- Открылось модальное окно с заголовком «Район (создание)».

Шаг 4:
- Поле ввода «Название района» и «Номер в списке»:
  - Присутствует
  - Активно (можно ввести текст)
  - В поле Номер в списке по умолчанию проставлен следующий номер записи 
- Кнопка «Внести»:
  - Присутствует
  - Активна всегда, подсвечена
- Кнопка «Отмена»:
  - Присутствует
  - Активна всегда, не подсвечена
- Иконка закрытия (крестик):
  - Присутствует в левом верхнем углу
- Кнопка «Добавить запись»:
  - Присутствует в левом углу таблицы Параметры
- Кнопка «Поиск»:
  - Присутствует в правом углу таблицы Параметры
- Кнопка «История»:
  - Присутствует в правом углу таблицы Параметры
- Кнопка «История»:
  - Присутствует в правом углу таблицы Параметры
- Чек-бокс:
  - Присутствует в строке наименования колонок таблицы Параметры
- Колоноки Парметр, Значение, Дата начала, Дата окончания таблицы Параметры:
  - Присутствует в таблице Параметры
- Строка пагинации:
  - Присутствует в нижнем левом углу

Шаг 5:
- При пустом поле ввода кнопка «Внести» кликабельна (зеленого цвета), кнопка «Отмена» кликабельна (не выделена цветом).
- При пустом поле и наличии сообщения «Поле не может быть пустым» кнопка «Внести» некликабельна, кнопка «Отмена» кликабельна (не выделена цветом).

Шаг 6:
- Кнопка «Добавить запись»:
  - Присутствует всплывающая подсказка «Добавить запись»
- Кнопка «Поиск»:
  - Присутствует всплывающая подсказка «Поиск»
- Кнопка «История»:
  - Присутствует всплывающая подсказка «История»

Шаг 7:
- При нажатии «Отмена» — окно закрывается, изменений в таблице нет.
- При нажатии на крестик — окно закрывается, изменений в таблице нет.

Шаг 8:
- При клике в поле «Название» и «Номер в списке» появляется рамка подсветка.
- Курсор меняется на текстовый (I-beam).

Шаг 9:
- При наведении на поля ввода — курсор текстовый (I-beam)
- При наведении на кнопку «Внести»,«Отмена»,Иконка закрытия,кнопки,чек бокс и колонки таблицы Параметры — курсор указатель (рука)
- При наведении на статичный текст — курсор стрелка
- При наведении на неактивные элементы — курсор стрелка 

Шаг 10:
- При клике на кнопку «Добавить запись» открывается окно «Виды параметров»

Шаг 11:
- При клике на кнопку «Поиск» открывается поле ввода с плейсхолдером «Поиск»
- Курсор меняется на текстовый (I-beam).

Шаг 12:
- При клике на кнопку «История» открывается список парметров отправленый в историю ранее

Шаг 13:
- Чек-бокс отображается.
- По умолчанию галочка не установлена
- При установке галочки  - галочка появляется / чек-бокс выделен цветом

Шаг 14:
- При клике на кнопку насторек табоицы открывается выпадающий список с полями
  - Обновить таблицу,
  - Настроить столбцы, 
  - Экспорт

Шаг 15:
- При наведении курсора в правый угол наименования колонок таблицы 
  - Параметр
  - Дата начала 
  - Дата окончания 
появляется стрелка фильтрации содержимого, направлена вниз
- При клике стрелка меняет положение на противоположное

Шаг 16:
- При клике на число отображаемых строк открывается выпадающий список с указанием кол-ва отображаемых строк на странице
- По умолчанию установлено 100
- При выборе другого значения, установлено в поле, отображается, подчеркнуто зеленой линией

Шаг 17:
- Присутствует статичная запись «Отсутствуют данные» в таблице Параметры и в строке пагиниции «Записей 1-0 из 0»
