# Тест-кейс 02: Добавление района с пустым именем (негативный)


**Приоритет:** High  
**Тип тестирования:** Нефункциональное  
**Окружение:** Windows 10, Yandex 25.10.3.1021, 1920x1080  
**Автор:** Антропова Ольга

**Предусловие:**  
Пользователь авторизован с 
Логин: DEMOWEB 
Пароль: awdrgy,
 открыт раздел «Адреса проживающих».

**Шаги:**
1. Нажать иконку «Плюс» в тулбаре таблицы.
2. В выпадающем списке выбрать «Район».
3. В открывшемся окне оставить пустым название района.
4. Нажать кнопку «Внести».

**Ожидаемый результат:**
- Окно не закрывается.
- Поле Название района подсвечено красным.
- Появилось сообщение "Поле не может быть пустым".
